# Feedback survey

If you have a moment, please fill out this short Google form. The
responses help us understand whether NixNan-style instrumentation
traces are useful for diagnosing the kind of issue you reported:

https://docs.google.com/forms/d/e/1FAIpQLSe6crMdlrEKOtUdwfLiXj54T3p-LzVhzYODsOgp7KHYITbIqQ/viewform?usp=publish-editor

The form is short. Even partial responses (a single answer) are
useful — we are trying to gauge whether bundling traces, kernel
attribution, and a tentative recommendation alongside a PyTorch bug
report shortens the time-to-cause for the people who file these
issues.
